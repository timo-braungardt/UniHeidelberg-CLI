# UniHeidelberg-CLI
No more annoying websites for operation-critical information like the mensa-menue.

## Setup
To get the modules, you have to run
```
git submodule update --init --recursive
```

To update submodules you have to run
```
git submodule update --remote --merge
```

## Modules
To see how to use the different scripts, read the readme of the submodule.

[Mensa Alert](./mensa_alert/README.md)  
Look up the daylie menue of today or for the whole week.

[Uni Heidelberg VPN](./uni_heidelberg_vpn/README.md)  
Connect to the Uni VPN without the GUI. 
It can also fetch your credentials from a Keepass file.

[Uni Heidelberg Campus Card](./UniHeidelberg-Campuscard/README.md)  
Check the balance on your card or list all your purchases and see some analysis.
